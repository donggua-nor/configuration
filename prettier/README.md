# `@donggua/prettier-config`

## Usage

#### Install

```bash
$ ${ur pkgManager} add @donggua/prettier-config -D
```

#### Configure

- `package.json`

```json
{
  // ...
  "prettier": "@donggua/prettier-config"
}
```

- `prettier.config.js` OR `.prettierrc.js`

```js
module.exports = require('@donggua/prettier-config')
```
